# Test Scenarios of SauceDemo

**Note:** Test scenarios can be further split or grouped based on business or management decisions in a real world example.

**Note:** Test scenarios are prepared according to the standard_user login.

### TS-01

- Verify that users can log in using valid credentials.

### TS-02

- Verify that login is rejected when invalid credentials are provided.

### TS-03

- Verify that login is rejected when required credentials are missing.

### TS-04

- Verify that an appropriate error message is displayed when login fails.

### TS-05

- Verify that the password input masks entered characters on the login page.

### TS-06

- Verify that users can complete the login process using keyboard only navigation.

### TS-07

- Verify that a logged in user can log out of the application.

### TS-08

- Verify that the user is redirected to the login page after logout.

### TS-09

- Verify that can't access the application after logout using the browser browse back.

### TS-10

- Verify that the navigation bar is visible and present.

### TS-11

- Verify that the navigation bar can be opened and closed.

### TS-12

- Verify that users can navigate to available pages using the navigation bar.

### TS-13

- Verify that the logout option is available in the navigation bar.

### TS-14

- Verify that a list of products is displayed on the Products page.

### TS-15

- Verify that product details including name, price, image, and description are displayed.

### TS-16

- Verify that products have action buttons available.

### TS-17

- Verify that products can be added to the shopping cart from the Products page.

### TS-18

- Verify that products can be removed from the shopping cart from the Products page.

### TS-19

- Verify that added items on the Products page retain their state after page refresh or navigating away to a different page.

### TS-20

- Verify that the product sorting function is available.

### TS-21

- Verify that products can be sorted alphabetically from A to Z.

### TS-22

- Verify that products can be sorted alphabetically from Z to A.

### TS-23

- Verify that products can be sorted by price from low to high.

### TS-24

- Verify that products can be sorted by price from high to low.

### TS-25

- Verify that the product list is updated according to the selected sorting option.

### TS-26

- Verify that users can open a selected item's page from the Products page.

### TS-27

- Verify that product details including name, price, image, and description are displayed on the item page.

### TS-28

- Verify that a product can be added to the shopping cart from the item page.

### TS-29

- Verify that a product can be removed from the shopping cart from the item page.

### TS-30

- Verify that the item page provides an action button for the product.

### TS-31

- Verify that users can navigate back to the Products page from the item page.

### TS-32

- Verify that an item added from the Item page retains its state after page refresh or navigating away to a different page.

### TS-33

- Verify that the shopping cart icon is visible and accessible.

### TS-34

- Verify that clicking the shopping cart icon navigates the user to the Shopping Cart page.

### TS-35

- Verify that the shopping cart icon displays the correct number of items.

### TS-36

- Verify that the shopping cart displays the correct products.

### TS-37

- Verify that users can view the contents of items added to the shopping cart.

### TS-38

- Verify that users can continue shopping from the Shopping Cart page.

### TS-39

- Verify that users can remove items from the shopping cart page.

### TS-40

- Verify that the Checkout button is available on the Shopping Cart page.

### TS-41

- Verify that Shopping cart page retains the state of the product added to the shopping cart after refreshing the page.

### TS-42

- Verify that the checkout process can be started when at least one item exists in the shopping cart.

### TS-43

- Verify that the checkout process can't be started when there is no item in the cart.

### TS-44

- Verify that user information is required to proceed with checkout.

### TS-45

- Verify that user is able to proceed to checkout after providing valid informations.

### TS-46

- Verify that checkout progression is blocked when required information is missing.

### TS-47

- Verify that an appropriate error message is displayed when required checkout information is missing.

### TS-48

- Verify that checkout process can be canceled anytime.

### TS-49

- Verify that an order overview is displayed before checkout completion.

### TS-50

- Verify that users can view item details, payment information, shipping information, and price information including tax during checkout.

### TS-51

- Verify that the checkout process can be completed successfully.

### TS-52

- Verify that a confirmation message is displayed after successful checkout.

### TS-53

- Verify that users can return to the Products page after checkout completion.

### TS-54

- Verify that Sshopping cart is reseted after checkout completion.

### TS-55

- Verify that checkout process pages don't get broken after refresh action.

### TS-56

- Verify that Products page has a page title.

### TS-57

- Verify that Shopping Cart page has a page title.

### TS-58

- Verify that first step of checkout process Checkout: Your Information has a page title.

### TS-59

- Verify that second step of checkout process Checkout: Overview has a page title.

### TS-60

- Verify that third step of checkout process Checkout: Complete has a page title.
